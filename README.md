# 🛫Welcom to Trip:ter

오직 나를 위한 여행,  믿고 맡길 수 있는 최고의 가이드 Trip:ter<br/>
트립터는 맞춤형 여행가이드 서비스 플랫폼으로 여행자들의 취향과 가치에 맞춘 여행을 제시하여<br/>
여러분의 소중한 순간이 행복한 추억으로 남을 수 있도록 최고의 가이드가 되어 함께 합니다.
